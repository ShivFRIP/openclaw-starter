# Task Commitments — Active Promises to Users

> **This file is your accountability ledger.** Every time you tell a user "I'll do X" or "working on X" or "yes, doing that now", you MUST log it here BEFORE starting the work. The heartbeat checks this file — stale commitments get flagged to the user.

## Format

```markdown
### [COMMIT-NNN] Short description
- **Promised to:** [user name / Telegram ID]
- **Promised at:** YYYY-MM-DD HH:MM IST
- **What was promised:** [exact commitment — what you said you'd do]
- **Expected output:** [file path, message, or deliverable]
- **Status:** ACTIVE / DONE / FAILED / PARTIAL
- **Completed at:** [timestamp, or blank if not done]
- **Verified:** [Yes/No — did you confirm the output exists and is correct?]
```
