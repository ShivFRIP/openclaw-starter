# Request Templates

These templates help you get the best results from your AI assistant. Copy, fill in the blanks, and send!

## 1. Research Request

Use this when you need research on a policy topic:

```
Research request:
Topic: [what you need researched]
Scope: [India-specific / international / both]
Timeframe: [recent week / month / year / all time]
Output: [summary / full brief / bullet points / data points only]
Project: [your-project-name]
```

**Example:**
```
Research request:
Topic: Recent changes to Extended Producer Responsibility rules
Scope: India-specific
Timeframe: Last 3 months
Output: Summary with key takeaways
Project: solid-waste-mgmt
```

## 2. Document Draft

Use this when you need something written:

```
Draft request:
Type: [policy brief / memo / analysis / letter / executive summary / presentation outline]
Topic: [subject]
Audience: [internal team / government officials / general public / donors]
Length: [short ~500 words / medium ~1500 words / detailed ~3000 words]
Key points to cover: [list any specific points]
```

**Example:**
```
Draft request:
Type: Policy brief
Topic: Impact of import duty reduction on solar panel manufacturing
Audience: Government officials
Length: Medium (~1500 words)
Key points to cover: Current tariff rates, domestic manufacturing capacity, employment impact
```

## 3. Knowledge Base File Lookup

Use this to find files in your shared knowledge base:

```
Find in KB:
Project: [project-name]
Looking for: [describe what you need]
```

## 4. Quick Analysis

Use this for comparisons or data interpretation:

```
Analyze:
Data/Topic: [what to analyze]
Question: [specific question you want answered]
Format: [table / bullet points / paragraph / chart description]
```

## 5. Document Review

Use this when you have a draft and want feedback:

```
Review this document:
[paste your text or describe the file]

Check for:
- Factual accuracy
- Argument strength
- Missing perspectives
- Clarity and readability
```

## Tips

- You don't HAVE to use these templates — just ask naturally
- Templates help when you want structured, detailed output
- Always specify the project name so outputs get saved to the right folder
